Apresentando um Poderoso Mecanismo de Rede Socialde Codigo Aberto

Elgg é um software de código aberto de rede social desenvolvido e hospedado no Github. Ele oferece um espaço de blog, comunidades com fóruns de discussões ou blogs comunitários, espaço para repositório de arquivos, e-portifólio, tecnologia RSS para o conteúdo gerado dentro da rede, entre outras coisas.

API principal bem documentada que permite aos desenvolvedores iniciar seu novo projeto com uma curva de aprendizado simples

O Composer é o gerenciador de pacotes de escolha que simplifica muito a instalação.

Sistema de controle de acesso que permite que os aplicativos criem diferentes niveis de acesso a conteúdo, bem como criem redes e intranets privadas

Grupos - suporte pronto para uso para grupos de usuários

Serviço de notificações que permite que os aplicativos inscrevam os usuários em notificações no local e por e-mail e implementem integrações com outros serviços terceirizados

Intranet é uma rede de computadores para uso privado de uma organização específica que utiliza os mesmos protocolos de comunicação da internet.
